## 1. FFT 진동 분석

### 분석 로그
- 2026-05-04_18-22-21.bin (노치 필터 적용 전)
- 2026-05-04_19-02-37.bin (노치 필터 적용 후)

### 분석 환경
- 배치 샘플러: ACC/GYR ~2000Hz, 총 469K 샘플
- 연산 방법: Welch PSD (NFFT=8192)

### 주파수 구조

| 피크 | 주파수 | 해석 |
|------|--------||------|
| 1차 | ~49 Hz | 모터 회전 기본 주파수 2,940 RPM |
| 2차 | ~98 Hz | 블레이드 패스 (2-blade x 49Hz) - 최대 진폭 |
| 3차 | ~146 Hz | 3배 하모닉 |
| 4차 | ~194-196 Hz | 4배 하모닉 |

### 모터 역산 검증
유효 전압 = 2,940 RPM / 150KV = 19.6V
스로틀 = 19.6V / 50.4V = 39%
-> 표준 이류중량 19kg에서 39% 스로틀 호버 -> 정상

## 2. 하모닉 노치 필터 파라미터 (확정)

INS_HNTCH_ENABLE = 1
INS_HNTCH_MODE   = 3
INS_HNTCH_FREQ   = 40
INS_HNTCH_HMNCS  = 3
INS_HNTCH_BW     = 20
INS_HNTCH_ATT    = 40

### 적용 근거
- HMNCS=3: 1st + 2nd 하모닉 동시 억제
- MODE=3: ESC DroneCAN RPM 기반 실시간 추적
- FREQ=40: 하한 가드

### FTN1 추적 결과
- PkAvg mean: 99.7 Hz
- FtX mean: 93.1 Hz
- 하한 가드(40Hz) 이하 이탈 없음

## 3. Loiter 파라미터 (확정)

LOIT_SPEED     = 250
LOIT_ACC_MAX   = 250
LOIT_ANG_MAX   = 15
LOIT_BRK_ACCEL = 150
LOIT_BRK_DELAY = 0.3
LOIT_BRK_JERK  = 500

### 튜닝 결정 근거
- LOIT_ANG_MAX=15: 건물 외벽 근접 + 200bar 수압 반력 환경에서 과도 기울기 방지
- LOIT_BRK_ACCEL=150: 호스 장력 밀림 억제 확인
- LOIT_BRK_JERK=300->500: 펌웨어 하한 위반 수정

- 건물 근접: LOIT_ANG_MAX 엄격 제한 필수
- ESC: PWM + DroneCAN -> MODE=3 RPM 텔레메트리 정상 수신
